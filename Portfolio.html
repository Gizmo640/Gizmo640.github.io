Project 1

The goal of this project is to build and test prototype of a variable-pitch drone propeller system. A variable pitch system is one where the angle of the blades can be controlled by the pilot. These systems are already being used in helicopters to make taking off and landing softer and more gradual. This project is about taking that principle and adding it to a drone for the same reasons, to make take-off and landing more gradual and less jerky.
